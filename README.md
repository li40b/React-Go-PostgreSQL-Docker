# React-Go-PostgreSQL-Docker

### 開発環境
|          |                   | 
| -------- | ----------------- | 
| Docker   | 24.0.6            | 
| golang   | 1.21.4-alpine3.18 | 
| postgres | 15.1              | 
| node     | 21.2-alpine3.18   | 
## frontendのコンテナに入るコマンド

`docker exec -i -t frontend sh`

## backendのコンテナに入るコマンド

`docker exec -i -t backend sh`
